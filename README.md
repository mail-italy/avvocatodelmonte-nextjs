
# Studio Legale Del Monte — versione locale Next.js

## Avvio rapido

```bash
npm install
npm run dev
```

Poi apri:

```bash
http://localhost:3000
```

## Contenuto del pacchetto

- homepage premium con focus su Avv. Federica Del Monte, avvocato cassazionista a Roma
- pagine servizio:
  - cassazione civile
  - cassazione penale
  - responsabilità medica
  - risarcimento danni
  - risarcimento danni da incidente stradale grave
  - infortuni sul lavoro
  - successioni complesse
- pagina Studio
- sezione Approfondimenti con articoli e pagine dinamiche
- pagina Contatti
- pagina Privacy
- sitemap e robots
- JSON-LD base
- form contatti con allegato per test locale
- pulsante Chiama con logica oraria lato client
- WhatsApp sempre visibile

## Nota sulla versione locale

Il form in localhost restituisce una conferma tecnica e non invia email reali.
Prima della pubblicazione online sarà sufficiente collegare il canale effettivo di ricezione delle richieste.

## Immagini

Le immagini del progetto sono già presenti in:

`public/images`

e possono essere sostituite facilmente mantenendo gli stessi nomi file oppure aggiornando i riferimenti nelle pagine e nei componenti.
